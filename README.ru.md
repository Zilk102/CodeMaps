# CodeMaps

**CodeMaps** — это мощный инструмент для визуализации архитектуры кодовой базы и предоставления архитектурного контекста AI-ассистентам. Приложение строит "живой" граф зависимостей вашего проекта, помогая разработчикам и нейросетям лучше понимать структуру, связи и технический долг.

## 🚀 Ключевые возможности

- **Автоматическое построение графа**: Глубокий семантический анализ кода (через AST и tree-sitter) для выявления связей между файлами, классами и функциями.
- **Интеграция с Model Context Protocol (MCP)**: CodeMaps выступает как сервер MCP, позволяя вашим AI-помощникам (например, Trae, Claude или Cursor) "видеть" архитектуру проекта, анализировать зависимости и предсказывать последствия изменений.
- **Архитектурная аналитика**:
  - **Blast Radius (Радиус поражения)**: Оценка того, на что повлияет изменение конкретного модуля.
  - **Health Score (Индекс здоровья)**: Выявление циклических зависимостей и архитектурных антипаттернов.
  - **Security Scanner**: Базовый поиск потенциальных уязвимостей в структуре кода.
- **Локальная работа**: Весь анализ происходит локально на вашей машине, исходный код никуда не отправляется.

### Интерактивный граф кода
Визуализация всей кодовой базы в виде интерактивного графа с файлами, классами, функциями и их связями.

### MCP Server Settings — Обзор
CodeMaps запускает MCP-сервер с HTTP и WebSocket endpoints, предоставляя AI-агентам структурированный доступ к архитектуре вашего проекта.

Более 10 встроенных инструментов для AI-агентов: анализ проекта, получение контекста графа, поиск узлов, обнаружение паттернов, сканирование безопасности и многое другое.

## 📦 Установка

Скачайте последнюю версию для вашей платформы из раздела [Releases](https://github.com/Zilk102/CodeMaps/releases).

Приложение не требует сложной установки и работает сразу на **Windows**, **Linux** и **macOS**.

### Быстрый старт

```bash
# Скачайте последний релиз для вашей платформы
# Windows: CodeMaps-x.x.x-win-x64.exe
# Linux: CodeMaps-x.x.x-linux-x86_64.AppImage (или .deb, .rpm)

# Запустите — установка не требуется!
```

## 🛠️ Разработка

Проект построен на стеке: **Electron + React + TypeScript + Vite**. Для визуализации графов используется Sigma.js.

### Требования

- Node.js 20+
- npm или yarn

### Запуск локально

```bash
# Клонирование репозитория
git clone https://github.com/Zilk102/CodeMaps.git
cd CodeMaps

# Установка зависимостей
npm install

# Запуск приложения в режиме разработки
npm run dev
```

### Сборка

```bash
# Создание production-сборки (Portable)
npm run build:portable

# Или сборка всех форматов
npm run build
```
