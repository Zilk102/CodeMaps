# Политика безопасности

Мы серьезно относимся к безопасности проекта CodeMaps. Поскольку инструмент анализирует исходный код локально, важно обеспечивать надежную защиту от внедрения вредоносного кода или утечек данных через зависимости.

## Поддерживаемые версии

Обновления безопасности выпускаются только для последней актуальной версии проекта.

| Версия | Поддерживается     |
| ------ | ------------------ |
| 1.0.x  | :white_check_mark: |
| < 1.0  | :x:                |

## Сообщение об уязвимости

Пожалуйста, **НЕ** сообщайте об уязвимостях в публичных GitHub Issues. Это может поставить под угрозу безопасность других пользователей до выхода исправления.

Вместо этого:

- Используйте функцию **GitHub Security Advisories** в этом репозитории для приватного отчета.
- Либо свяжитесь с мейнтейнером напрямую по электронной почте.

Мы подтвердим получение вашего отчета об уязвимости и будем регулярно информировать вас о статусе исправления.
